XIPs

XDC Network Improvement Proposals (XIPs) describe standards for the XDC platform, including core protocol specifications, client APIs, and contract standards.

Contributing

First review XIP-1. Then clone the repository and add your EIP to it. There is a template XIP here. Then submit a Pull Request to XDC Community's XIPs repository.
